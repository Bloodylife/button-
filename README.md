# Master Note Synthesizer

A modern, professional website for the Academic Knowledge Synthesizer tool that helps students compile their study materials into comprehensive master notes.

## Features

- **Multi-Format Support**: Upload PDF, PPTX, DOCX, textbooks, and scanned notes
- **Syllabus Alignment**: Every concept mapped to your syllabus structure
- **Smart Cross-Referencing**: Information merged intelligently without redundancy
- **Tables & Diagrams**: Charts and visual content preserved with proper captions
- **Academic Quality**: Clear language suitable for undergraduate to graduate level
- **Print-Ready PDF**: Professionally formatted output with table of contents

## Getting Started

Simply open `index.html` in your browser to use the application.

## Files

- `index.html` - Main HTML structure
- `styles.css` - Modern CSS styling with responsive design
- `script.js` - Interactive JavaScript functionality

## How It Works

1. **Upload Your Syllabus** - Start by uploading your course syllabus
2. **Add Study Materials** - Upload up to 50 study materials in various formats
3. **Download Master Notes** - Get a comprehensive, publication-ready PDF
